# Reglas de Negocio

- El usuario puede obtener la informacion del clima actual
- La consulta puede hacerse por ciudad
- La consulta tambien puede hacerse por coordenadas
- Si la consulta es por ciudad, primero se deben obtener las coordenadas
- Si la consulta es por coordenadas, latitud y longitud son obligatorias
- La latitud debe estar entre -90 y 90
- La longitud debe estar entre -180 y 180
- La app debe devolver respuestas simples y en espanol
- La descripcion del clima debe ser legible para el usuario
